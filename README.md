# electronics-555-timer-rev1-0
A single-board solution to the three operating modes of the 555 timer (astable, monostable, and bistable).

Jumpers to bridge for each operating mode:
Astable: 2,4,5;
Monostable: 1,3,4;
Bistable: 1
